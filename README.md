# DreamGuide

DreamGuide — мобільний застосунок для покращення якості сну.  
Мета проєкту — допомогти користувачам відстежувати сон, аналізувати його якість та отримувати персональні рекомендації.

## Основні можливості
- Трекінг сну за допомогою смартфона  
- Smart-будильник  
- Темна тема  
- Аналітика (графіки, статистика сну)  
- Персональні поради для покращення сну

## Технології
- Kotlin / Android Studio  
- Firebase (аналітика)  
- ML-модуль для Sleep Score
